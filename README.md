# ThenUIKit



针对基础THEN UIKit框架进行封装, 主要通过.then链式使用



## Installation

#### Cocoapods

```
pod 'ThenUIKit', :git => 'https://github.com/ghostcrying/ThenUIKit.git'
# , :tag => '1.0.0'
# 可根据tag指定使用
```



#### Carthage

```
github "ghostcrying/ThenUIKit"
```



#### Swift Package Manager

```
https://github.com/ghostcrying/ThenUIKit.git
```
